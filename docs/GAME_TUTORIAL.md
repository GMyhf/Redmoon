# CRIMSON RELAY 玩家教程

## 一、开始游戏

CRIMSON RELAY 是一款浏览器在线动作 RPG。进入游戏后输入角色名，选择职业并加入灰港中继站。角色名会绑定账号进度；重新进入时需要使用浏览器保存的会话凭据。不要把会话令牌或恢复码发给其他人。

### 基本操作

| 操作 | 方式 |
| --- | --- |
| 移动 | 左键点击地面，或使用 `WASD` / 方向键 |
| 锁定并攻击 | 左键点击敌人 |
| 手动攻击 | 右键朝准星方向攻击，空格也可攻击 |
| 使用技能 | `Q`、`E`、`R`、`C`、`F` |
| 使用修复药剂 | `V`，使用背包中的第一瓶药剂 |
| 自动战斗 | `T` 开关 |
| 奔跑 | 移动时使用冲刺控制 |

手动移动会取消当前点击移动指令。服务器负责判定位置、命中、伤害、经验和奖励，客户端显示服务器结果。

## 二、职业与战斗

游戏提供八个原创职业。选择职业时先看自己喜欢的节奏：先锋偏力量和生存，逐风者偏敏捷，导能者偏精神与技能，壁垒者偏防御，月刃和蚀星等职业则提供更特殊的技能组合。职业一旦绑定到角色名，之后不能用另一职业覆盖同名角色。

战斗时先观察敌人的等级、血量和攻击距离。近战职业应利用移动接近目标，远程职业要保持距离；敌人攻击前通常会有动作和范围提示。城镇安全区不会受到怪物伤害，适合整理装备、购买药剂和恢复状态。

## 三、升级、属性与技能

击杀敌人会获得经验。达到升级所需经验后，角色升级并获得属性点和技能点。属性与职业成长方向相关：力量提高近战输出，敏捷强化灵活与攻击节奏，精神强化技能路线，体质提高生存能力。

新手可以开启自动加点，让服务器按职业权重分配升级点；想自己规划时，可在界面关闭自动加点，再手动升级属性和技能。技能有等级和解锁等级，先确保角色等级达到要求，再把技能点投入常用技能。

等级上限为 1000。达到 10 级后可以转生：等级与当前经验归一，但换取永久属性点、生命上限和伤害加成。转生适合已经熟悉职业、准备长期培养角色的玩家，不建议刚达到门槛就盲目操作。

## 四、地图、任务与装备

灰港中继站通过传送门连接九张主题猎场。各地图敌人等级大致如下：

| 地图 | 敌人等级 | 建议用途 |
| --- | ---: | --- |
| 暮居街区 | 1-25 | 新手起步、前两环任务 |
| 旧都核心 | 15-45 | 低等级过渡 |
| 北境回山 | 40-110 | 中前期升级 |
| 锈蚀废料场 | 90-210 | 中期升级与装备积累 |
| 赤潮沙海 | 180-360 | 中高等级猎场 |
| 霜脊山线 | 330-520 | 高等级过渡 |
| 坠落城堡 | 480-680 | 后期猎场与副本主题区域 |
| 失落星港 | 650-860 | 高等级挑战 |
| 悬空天城 | 820-1000 | 满级前后的终局区域 |

城镇敌人等级为 1-18，主要用于新手距离曲线，不适合作为长期刷怪区域。地图等级带存在重叠，例如 15-25 级可以继续留在暮居，也可以尝试旧都；不要只看地图名称，进入前应比较自己的等级、装备和药剂储备。等级较高的敌人通常拥有更高生命、伤害和经验，精英与 Boss 还会在所属地图等级基础上进一步强化。

八环任务链会从清除裂隙体开始，逐步要求讨伐暮牙兽、精英怪、区域 Boss 和高等级物种，最后的“深红的终局”可以重复完成。任务进度按击杀类型记录，完成后获得经验、金币和复苏露。

敌人可能掉落普通装备、精制、稀有、史诗装备、遗物以及特殊掉落。装备覆盖武器、盾牌、头盔、项链、胸甲、腰带、手套、裤子、靴子和三枚戒指，共十二个穿戴位。优先比较装备等级要求、稀有度和强度评分；不想手动整理时可开启自动换装。

城镇商店提供修复药剂、定制装备和遗物匣。药剂可以在战斗中使用，复苏露可用于原地复活，也可在黑市商人处兑换遗物匣。金币主要来自击杀、任务和出售背包物品。

## 五、组队打怪

在社交面板邀请在线玩家，对方在事件栏接受后即可组队，队伍最多四人。普通地图击杀经验按以下方式结算：击杀者获得怪物经验的 100%，每名符合条件的队友获得 60%，不是队伍内均分。

队友要获得共享经验，必须与怪物处于同一张地图、距离怪物不超过 1200px、保持存活并处于已认证的在线状态。跨地图、断线保席、尚未完成认证或已经阵亡的成员不会获得该次共享经验。符合范围的队友也会获得对应任务计数。

队伍协作建议：先锋负责吸引近战压力，导能者和远程职业从侧后方输出，所有人保持在共享范围内；击杀 Boss 前先确认队友没有跑到别的地图。队伍频道适合发布目标和撤退信息，离队或断线会自动清理队伍关系。

## 六、深红中继密库

队伍首位成员可以在所有队员同图、存活且尚未进入副本时开启“深红中继密库”。副本根据队伍平均等级生成固定阵容：五名密库守卫和一名密库监察者首领。阵容可复现，但等级会随队伍成长。

副本实例最多同时存在 32 个，15 分钟未完成会超时并把成员送回中继站。正常在城镇重生会退出副本；在战斗中使用复苏露原地复活，则可以继续当前副本。副本完成奖励按队伍成员集合结算一次，包含经验、金币和复苏露，不要重复进出期待重复领取。

## 七、推荐成长路线

1. 在灰港熟悉移动、锁定、技能和药剂操作。
2. 先完成前两环任务，利用掉落装备替换低强度部位。
3. 开启自动战斗或自动加点前，先确认自己理解它们的作用；手动操作更适合 Boss 战。
4. 与同等级玩家组队，保持同图和 1200px 共享范围，连续清理任务目标。
5. 达到 10 级后，根据长期方向决定是否转生。
6. 组齐队伍并整理药剂后进入深红中继密库，预留完整的十五分钟战斗时间。
7. 进入高等级地图前，检查装备等级、药剂数量、技能冷却和复苏露储备。

遇到异常时，先回到城镇确认连接状态，再检查角色是否仍在正确地图；不要反复刷新或把会话凭据交给他人。
