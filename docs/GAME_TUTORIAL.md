# CRIMSON RELAY 玩家教程

《深红中继（CRIMSON RELAY）》是一款服务器权威的浏览器在线动作 RPG。你在灰港中继站创建角色，穿过十道传送门进入九张主题猎场与一张开放 PvP 的战斗区，一路从 1 级打到 1000 级。**所有战斗结果——命中、伤害、经验、掉落、位置——都由服务器判定，客户端只负责显示。** 客户端里没有任何游戏规则，因此改客户端既改不动数值，也无法作弊。

---

## 一、开始游戏

进入游戏后输入角色名，选择职业并加入灰港中继站。

- 角色名会**绑定账号进度**：重新进入时用浏览器保存的会话凭据（session token / 恢复码）继续。**不要把会话令牌或恢复码发给任何人**——它等同于你的账号。
- 职业一旦绑定到角色名就**无法用另一职业覆盖同名角色**，创建前想清楚。
- 受保护的角色名在别处登录会被服务器拒绝（`NAME_IN_USE` / `INVALID_TOKEN`）。

### 基本操作

| 操作 | 方式 |
| --- | --- |
| 移动 | 左键点击地面，或使用 `WASD` / 方向键 |
| 锁定并攻击 | 左键点击敌人 |
| 手动攻击 | 右键朝准星方向攻击，`空格` 也可攻击 |
| 使用技能 | `Q`、`E`、`R`、`C`、`F` |
| 使用修复药剂 | `V`，使用背包中的第一瓶药剂 |
| 自动战斗 | `T` 开关 |
| 奔跑 / 冲刺 | 移动时使用冲刺控制 |

手动移动会取消当前的点击移动指令。近战靠移动贴近目标，远程靠走位保持距离。城镇安全区不会受到怪物伤害，适合整理装备、购买药剂和恢复状态。

## 二、八大职业

游戏提供**八个原创职业**（名字、背景与外观均为本作原创）。选择时先想清楚自己喜欢的节奏：偏力量与生存、偏敏捷机动、偏精神与技能，还是偏极限防御。下表的初始属性为**力量 / 敏捷 / 精神 / 体魄**。

| 职业(代号) | 名号 | 定位 | 初始属性 | 特点 |
| --- | --- | --- | --- | --- |
| 先锋 vanguard | 绯岚 | 重刃突击 | 6 / 3 / 2 / 7 | 高韧性近战，把冲势砸成震荡波 |
| 导能者 channeler | 汀兰 | 星能术师 | 2 / 4 / 7 / 4 | 远程场域控制，星能弹幕与巨球 |
| 逐风者 strider | 疾风 | 游侠 | 4 / 7 / 3 / 4 | 高机动，飞镖连射与位移追击 |
| 壁垒者 bulwark | 铁岳 | 壁垒守卫 | 7 / 2 / 2 / 8 | 极限生存，近距重锤与全域震荡 |
| 长弓 longshot | 星眸 | 狙击手 | 4 / 8 / 3 / 3 | 全场最远射程，贯穿长矢 |
| 焚灵 pyre | 焰蝶 | 烈焰术士 | 2 / 3 / 8 / 5 | 范围爆发，多圈烈焰清场 |
| 蚀者 eclipse | 玄晓 | 明暗行者 | 3 / 4 / 7 / 4 | 双线成长，光辉/深渊两套技能 |
| 月刃 moonblade | 月绫 | 月刃舞者 | 5 / 7 / 3 / 3 | 全场最快攻速，贴身收割 |

新手上手推荐：想稳、想扛，选**先锋**或**壁垒者**；想灵活输出，选**逐风者**或**月刃**；想远程安全刷怪，选**长弓**或**导能者**；喜欢范围清场，选**焚灵**。**蚀者（玄晓）机制最独特，建议熟悉基础后再尝试**（见下方「蚀者的双魂机制」）。

## 三、属性系统

升级会给你属性点，投入四种属性；每种属性都影响不同的战斗方向：

- **力量（power）**：提高近战系技能与普攻的输出。
- **敏捷（agility）**：强化灵活与攻击节奏，是敏捷系职业的伤害来源。
- **精神（spirit）**：强化技能路线的伤害，并**决定法力（MP）上限与回复**。
- **体魄（vitality）**：大幅提高生命上限（每点约 +11 生命），是生存的基石。

生命上限还随等级和装备增长；法力上限主要由精神决定，并随等级小幅增长、随时间自动回复（精神越高回得越快）。

### 魂力护障（Soul Barrier）

每名角色都自带**魂力护障**：受到伤害时，一部分会**优先从法力（MP）扣除，再扣生命**。也就是说，法力不仅用来放技能，也是一层“缓冲血条”。精神堆得高、法力池大的职业，实际生存能力比生命数字看起来更强；把法力耗光去硬刚，则会失去这层保护。蚀者的光辉线技能还能主动治愈并强化这层护障。

### 自动加点

新手可开启**自动加点**，让服务器按职业权重分配升级点（例如先锋偏力量与体魄，导能者偏精神）。想自己规划时，在界面关闭自动加点，再手动分配。

## 四、技能系统

每个职业有五个技能槽：`Q`、`E`、`R`、`C`、`F`。

- **`Q` / `E` / `F` 是职业专属技能**：`Q`、`E` 是主力输出/位移，`F` 是威力最大、冷却最长的终极技（如先锋「绯岚断空」、导能者「星潮涌落」、月刃「月蚀乱舞」）。
- **`R` / `C` 也是职业专属**，但两个槽位在全职业保持同一个**定位**：`R` 是重击（5 级解锁），`C` 是位移 + 环形爆发（10 级解锁）。各职业的形态和吃的属性都不同——例如长弓的 `R`「猎隼标记」是全场最远的单发贯穿，壁垒者的 `R`「震岳壁击」是最短射程的最重一击；而壁垒者的 `C`「守望铁环」**完全不位移**，用全场最大最密的环换掉了逃生手段。`R`/`C` 都按各职业的**主属性**成长，和 `Q`/`E`/`F` 一样值得投点。

技能有**等级**和**解锁等级**：先让角色达到解锁等级，再把技能点投入常用技能。技能等级越高，伤害、范围、弹数（贯穿数）等随之提升。技能伤害通常由固定值、每级成长和你的核心属性共同决定——所以属性和技能要一起养。

### 蚀者的双魂机制

蚀者（玄晓）的 `Q` / `E` / `F` 会根据**名誉（reputation）**在两条线之间切换：

- **光辉线**（名誉 ≥ 0）：贯穿圣枪、治愈并强化魂力护障、黎明环爆并回复法力。
- **深渊线**（名誉 < 0）：三向蚀寒弹、寒霜环爆、双重永夜冰环。

每次施法都会把名誉朝你**选定的归属**（光辉或深渊）推动两点，跨过零点就切换整套技能。蚀者还会随击杀积累**意志（will）**——一种一生只增不减的资源。选定一条线专精，通常比来回横跳更强。

## 四点五、装备精炼

站到**锻匠·坤铁**旁边，背包里稀有及以上的装备（已穿在身上的也行）会多出一个「炼」按钮。

- **代价**：每次尝试消耗意志与金币，随装备等级和当前阶数上升。意志来自击杀，一直在悄悄累积。
- **回报**：每成功一阶，该装备的全部数值放大 15%，满阶（+4）合计 **+60%**。
- **风险**：成功率随阶数递减 —— **90% → 70% → 50% → 30%**。**失败会掉 1 阶**（+0 时无阶可掉，只损失费用）。
- **门槛**：推到 **+3 需要 200 荣誉、+4 需要 400 荣誉**（见下一节）。前两阶对所有人开放——新掉的装备永远能改。**荣誉只是门槛，不会被扣掉**。
- **保险**：向**黑市商人·影三**用 1 复苏露换一张**护炉印**，按住 `Shift` 点「炼」即可为这次尝试启用；失败时它替你挡下掉阶，自身消耗。

从 +0 一路推到 +4，按上面的成功率平均需要约 **8 次**尝试。**不必每次都带护炉印**：+0→+1 有九成把握且失败无损，把复苏露省到 +2 以后的高阶再用，性价比高得多。

## 四点六、荣誉

**荣誉是你猎过什么的证明**，不是一种货币——它只被「检查」，从不被「花掉」。

- **来源**：击杀**精英**（+1）与 **Boss**（+5）。普通杂兵不给。上限 1000。
- **用途**：目前门禁装备精炼的最后两阶（+3 需 200、+4 需 400）。它要求的是「你真的去猎过」，而不是再刷一种资源。
- **分档**：无名（0）→ 闻名（200）→ 信重（400）→ 威名（600）→ 传颂（800）。

> 荣誉与蚀者的**名誉（reputation）**是两个不同的东西：名誉是玄晓自己选择的光辉/深渊构筑轴，符号翻转会切换他整套技能；荣誉是所有职业共有的社会声望。互不影响。

## 五、升级、转生

击杀敌人获得经验，达到升级所需经验后升级，获得属性点和技能点。等级越高的敌人生命、伤害和经验越高；精英与 Boss 在所属地图等级基础上进一步强化。

**等级上限为 1000。转生在 1000 级解锁**——它是终局，不是前期捷径：

- 代价：等级与当前经验归一（回到 1 级重打）。
- 回报：**永久属性点、生命上限加成、伤害加成**，且可无限叠加多次转生。

满级后经验不再累积，转生是唯一的前进方向。正因为每一轮转生都要付出一次完整的 1→1000 攀爬，它的回报才敢做成永久且不封顶。

## 六、地图与敌人

灰港中继站通过传送门连接**九张主题猎场**（外加第十道门通向战斗区「血斗回廊」，见 9.6 节），围绕城镇排成一圈，每张猎场配一座 Boss。各地图敌人等级与首领大致如下：

| 地图 | 敌人等级 | 区域 Boss（等级） | 建议用途 |
| --- | ---: | --- | --- |
| 暮居街区 | 1–25 | — | 新手起步、前两环任务 |
| 旧都核心 | 15–45 | — | 低等级过渡 |
| 北境回山 | 40–110 | 岩父（130） | 中前期升级 |
| 锈蚀废料场 | 90–210 | 锈王（240） | 中期升级与装备积累 |
| 赤潮沙海 | 180–360 | 棘颚兽（260）· 沙喉（380） | 中高等级猎场 |
| 霜脊山线 | 330–520 | 霜角（550） | 高等级过渡 |
| 坠落城堡 | 480–680 | 墓行者（700） | 后期猎场与副本主题区域 |
| 失落星港 | 650–860 | 舰魂（870） | 高等级挑战 |
| 悬空天城 | 820–1000 | 深红督军（1000） | 满级前后的终局区域 |

城镇（灰港）本身怪物等级为 1–18，仅用于新手距离曲线，不适合长期刷怪。

**地图等级带存在重叠**（例如 15–25 级既可留在暮居，也可试旧都）：不要只看地图名称，进门前先比较自己的等级、装备和药剂储备。贸然踏进高出自己太多的地图，一次照面就可能致命。

常见怪物种类（越靠后越强）：裂隙体、暮牙兽、烬翼、棘甲爬兽、岩角兽、霜眼祭兽、废钢巨兽、风暴浮眼、虚空吞喉。近战型会冲撞撕咬，远程型（烬翼、霜眼祭兽、风暴浮眼、虚空吞喉）会隔空放技能，攻击前通常有前摇提示，可提前走位规避。

### 任务链

**八环任务链**从清除裂隙体开始，依次要求讨伐暮牙兽、猎杀精英怪、击败区域 Boss（棘颚兽、沙喉）、讨伐高等级物种（岩角兽、虚空吞喉），最后一环「**深红的终局**」是击败深红督军，**可无限重复**。任务进度按击杀类型记录，完成后获得经验、金币和复苏露；奖励随任务阶段的内容等级放大，一路都有意义。

## 七、装备与稀有度

敌人可能掉落装备与特殊物品。装备覆盖**十二个穿戴位**：武器、护盾、头盔、项链、胸甲、腰带、手套、裤子、靴子，以及**三枚戒指**。

稀有度从低到高共七档，越高越强、越罕见：

| 稀有度 | 名称 | 说明 |
| --- | --- | --- |
| common | 普通 | 基础掉落 |
| fine | 精制 | 常见强化 |
| rare | 谐振 | 中档 |
| epic | 赤月 | 高档 |
| relic | 遗物 | 命名装备（如霜月刀·霁澜、凝神之戒、长明之链），大幅属性或独特攻击 |
| unique | 唯一 | 服务器级稀有掉落，全服同时在场数量有限 |
| sunset | 日落 | 最稀有的一档，需要一定等级才会掉落，出现即抢 |

选装备优先比较**装备等级要求、稀有度和强度评分**。不想手动整理时可开启**自动换装**，让服务器为你替换更强的部位。背包有容量上限，掉落物在地上有存在时限、拾取有半径，靠近时会自动飞向你。

## 八、城镇与经济

城镇里有三位商人（原创角色）：

- **杂货商·芦婆**：出售小/大修复药剂，价格与治疗量随你的等级成长，一路 1–1000 级都保持有用。
- **锻匠·坤铁**：出售定制装备（随机部位，精制以上品质）。
- **黑市商人·影三**：用**复苏露**兑换遗物匣（开出随机遗物）。

两种通货：

- **金币（gold）**：每次击杀都会掉，主要用于买药剂、定制装备；也来自任务和出售背包物品。
- **复苏露（dew）**：稀有掉落。可在战斗中**原地复活**（不必被送回城镇），也可在黑市换遗物匣。省着用。

## 九、组队打怪

在社交面板邀请在线玩家，对方在事件栏接受后即可组队，**队伍最多四人**。普通地图击杀经验按以下方式结算：

- **击杀者获得怪物经验的 100%**；
- **每名符合条件的队友额外获得 60%**（不是队伍内均分，而是各自加成）。

队友要获得共享经验，必须满足全部条件：

- 与怪物处于**同一张地图**；
- 距离怪物**不超过 1200px**；
- 保持**存活**；
- 处于**已认证的在线状态**。

跨地图、断线保席、尚未完成认证或已阵亡的成员，不会获得该次共享经验；符合范围的队友也会获得对应任务计数。

**协作建议**：先锋/壁垒者负责吸引近战压力，导能者、长弓、焚灵等从侧后方输出，月刃和逐风者绕后收割，所有人保持在 1200px 共享范围内；击杀 Boss 前先确认没有队友跑到别的地图。离队或断线会自动清理队伍关系。

## 九点五、决斗

在社交面板的「同图在线」里点某人的「决」发起挑战，对方**应战**后，你们会一起被送进一座**独立的竞技场**——除了你们俩，那里什么都没有。

- **双方同意才会开始**。收到挑战可以应战，也可以回绝；60 秒不理会自动作废。
- **输赢不影响任何进度**：没有经验、金币、掉落，也没有荣誉。倒下即结束，双方满血回到进场前站的地方。**输一场只输这场。**
- 竞技场外仍然打不到人。**只有在你亲自同意的这场决斗里，你的攻击才会落在另一个玩家身上。**
- 180 秒不分胜负判平局；也可以随时点「认输」结束。决斗期间不能开副本。

想验证自己的加点和装备到底行不行，这里是唯一不用付代价的地方。

## 九点六、血斗回廊（战斗区）

城镇门环的第十道门通向**血斗回廊**——**全服唯一一张任何人都能攻击任何人的地图**。走进那道门，就是同意。

- **为什么去**：它同时是猎场，等级带 300-1000，精英照常给荣誉。**荣誉的来源和风险在同一个地方。**
- **输了会怎样**：击杀者拿走你 **10% 金币**和**至多 10 点荣誉**。**装备一件不掉，经验一点不掉。**
- **荣誉只能被抢走你真正拥有的那部分**：荣誉为 0 的人被杀，对方一无所获。所以拿小号当靶子没有意义，两个朋友互喂也是零和——**想涨荣誉，只能去赢真正有声望的人**。
- 被**怪物**杀死不额外损失金币或荣誉，只有被玩家击杀才结算。
- 门外的一切照旧：城镇、九张猎场、副本里**永远打不到人**。

现在你能看见别人的荣誉了——分档就是给别人看的。回廊里，高荣誉既是勋章，也是靶子。

## 九点七、军团

军团是一支**比会话更长命**的队伍：你下线了，位置还在花名册上。

- **建立**：需要**等级 30** 与**荣誉 100**。荣誉和精炼一样**只是门槛，不会被扣掉**——它要求的是「你真的猎过」。团名全服唯一。
- **军衔**：统领 / 副官 / 团员。副官可以招募和逐出团员；谁也不能动与自己同级或更高的人。
- **统领不能一走了之**：要么把军团**转让**给别人（对方需要接掌，这是要约不是命令），要么**解散**。
- **招募要对方同意**：邀请只是邀请。
- **军团频道**：聊天框选「军团」，只有在线团员听得到。
- 你的军团名与军衔**别人看得见**。

## 十、深红中继密库（副本）

队伍首位成员可在**所有队员同图、存活、且尚未进入副本**时开启「**深红中继密库**」。

- 副本按**队伍平均等级**生成固定阵容：**五名密库守卫 + 一名密库监察者首领**。阵容可复现，但等级随队伍成长。
- 全服**最多同时存在 32 个**副本实例。
- **15 分钟未完成会超时**，把成员送回中继站。
- 在城镇正常重生会退出副本；在战斗中用**复苏露原地复活**，则可继续当前副本。
- 完成奖励按队伍成员集合**结算一次**（经验、金币、复苏露），不要反复进出期待重复领取。

进副本前预留完整的十五分钟战斗时间，并确认药剂、复苏露储备充足。

## 十一、推荐成长路线

1. 在灰港熟悉移动、锁定、技能和药剂操作，看清 `Q/E/R/C/F` 各自的作用。
2. 先完成前两环任务，用掉落装备替换低强度部位。
3. 开启自动战斗或自动加点前，先确认自己理解它们的作用；**手动操作更适合 Boss 战**。
4. 与同等级玩家组队，保持同图和 1200px 共享范围，连续清理任务目标。
5. 转生要到 1000 级满级才解锁，属于长线目标；前期不必为它规划。
6. 组齐四人、整理好药剂与复苏露后进入深红中继密库，预留完整的十五分钟。
7. 进入更高等级地图前，检查装备等级、药剂数量、技能冷却和复苏露储备；地图等级带重叠，宁可低打高练一环，也别越级送死。

遇到异常时，先回到城镇确认连接状态，再检查角色是否仍在正确地图；不要反复刷新，也不要把会话凭据交给任何人。
