# 逐职业伤害基线 · T-030

这是一份可复算的设计基线，不是承诺玩家最终 DPS。每个职业使用当前 `BASE_STATS`，技能按 `SKILL_BEHAVIORS` 的每个 projectile/burst 逐 tick 展开；不计命中数之外的移动收益、装备、暴击和敌方防御。

## 规则

单个投射物伤害为 `base + perLevel * skillLevel + Σ(stat * multiplier)`。burst 的总量乘以当次 count，fan 的总量乘以 angles 数量；多段技能分段相加。技能等级固定为 1，等级成长与装备留给后续基准工具。

## 审计结果

- 8 个职业的 R/C 都使用本职业的主加点属性，没有回退到 `shared:r`/`shared:c`。
- 每个职业的 R/C 都能在定义层展开为有限、非负的 tick 伤害；没有空行为、负数伤害或缺失技能键。
- R 是承诺较重的单段/窄线技能，C 是位移或范围技能；不能把二者的总命中数直接当作单目标 DPS 排名。
- 现有代码没有把“同一目标可同时吃到多少 burst 弹体”作为独立平衡参数，下一轮应以实际碰撞模拟补一组 20Hz、固定站位、无装备的单目标测量。

当前结论是“规则完整、未发现明显职业空洞”，不是“八职业已平衡”。
